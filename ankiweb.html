Say you have note1 (with the new cards 1,2,3) and note2 (with the new cards A,B). Anki will show you the cards in this order: 1,A,2,B,3 even if you have disabled "Bury related new cards until the next day". There is no option that allows you to show the cards in the order 1,2,3,A,B. This add-on allows you to disable this same-day spacing from the menu "Study". other authors - lovac42 - Thomas Kahn technical background Same-day spacing in Anki occurs because in the function _burySiblings there is code that is described as "if bury disabled, we still discard to give same-day spacing". compatibility - This add-on overwrites the built-in function "_burySiblings". It is not compatible to other add-ons that also overwrite/modify this function. As far as I see at the moment the only other add-on that changes this function is Bury related notes. - if future versions of Anki change the "_burySiblings" function you will run into problems.
